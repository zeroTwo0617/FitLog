# FitLog 训练记录本（微信小程序）

用结构化方式记录每一次训练的「动作-组数-次数-重量」，并提供训练计划、数据趋势、打卡日历与身体数据追踪。

- 形态：微信小程序原生 + 微信云开发（CloudBase）
- 文档：[开发文档.md](./开发文档.md)（PRD v0.1）

## 快速开始（开发环境）
1. 用「微信开发者工具」导入本目录
2. 开通云开发环境，填入 `env` 配置
3. 见 `开发文档.md` 的 Phase 0 ~ Phase 4 规划
